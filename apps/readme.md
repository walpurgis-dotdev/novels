## Installation Steps ğŸ› ï¸

1. **Clone the project**

   ```sh
   git clone git@github.com:lilwxs/__graduationThesis_VibeVerse.git
   ```

2. **Navigate to the project directory**

   ```sh
   cd __graduationThesis_VibeVerse
   ```

